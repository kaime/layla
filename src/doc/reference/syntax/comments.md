# Comments

Line (`//...`) and block (`/* ... */` comments are allowed. Both types are by default output as `/* ... */` on css.

## Line comments

``` lay-todo
// This is a line comment
```

```css-todo
/* This is a line comment */
```

## Block comments

```lay-todo
/**
 * This is a block comment
 */
```

```css-todo
/**
 * This is a block comment
 */
```

- Are not overwritable

  ~~~ lay
  green = #f00
  ~~~

  ~~~ ReferenceError
  ~~~

- Are case sensitive

  ~~~ lay
  color::names {
    i: steelblue
    ii: STEELBLUE
    iii: steelBlue
    iv: SteelBlue
  }
  ~~~

  ~~~ css
  color::names {
    i: #4682b4;
    ii: STEELBLUE;
    iii: steelBlue;
    iv: SteelBlue;
  }
  ~~~
